# Hello world

This is the **first post** on the blog. It uses frontmatter for metadata and markdown for content.

## Features

- Parsed with `gray-matter`
- Rendered with `remark` and `remark-html`
- Statically generated at build time

## Code

You can use inline `code` or fenced blocks:

```js
const greeting = 'Hello, blog!';
console.log(greeting);
```

## Links

[Next.js](https://nextjs.org) powers this app. Add more `.md` files in `/posts` to create new posts.
